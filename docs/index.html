🚀 快速开始

1

安装依赖

pip install -r requirements.txt
2

运行基础实验

python main.py --config configs/simple_mnist.yaml
3

查看结果

实验结果将在终端显示，包括各轮次的损失和准确率

架构特点

  • 模块化设计：各模块职责明确，便于扩展和维护
  • 抽象基类：使用ABC模式确保接口一致性
  • 配置驱动：基于YAML配置文件的灵活实验设置
  • 插件化支持：模型、数据集、聚合算法均可扩展
